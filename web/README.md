# 听写软件 - 前端版

一个纯前端实现的听写软件，无需后端也可完整运行。

## 功能特性

### 🎧 核心功能
- **听写练习**：选择组别进行听写练习
- **语音合成**：使用 Web Speech API 实现高质量语音朗读
- **暂停/继续**：支持听写过程中的暂停和继续
- **长词延长**：对长单词自动增加间隔时间

### 📁 数据管理
- **组别管理**：创建、编辑、删除词汇组别
- **数据持久化**：使用 localStorage 本地存储
- **导入导出**：支持 JSON 格式的数据导入导出

### 📷 文字识别
- **摄像头识别**：使用手机摄像头拍摄课本单词
- **图片上传**：支持上传图片进行识别
- **OCR 技术**：集成 Tesseract.js 实现前端文字识别

### 🔄 后台兼容
- **一键切换**：支持配置开启/关闭后台通信
- **API 调用**：兼容后端 REST API
- **数据同步**：支持与后台数据同步

## 技术栈

- **前端框架**：原生 HTML5 / CSS3 / JavaScript
- **语音合成**：Web Speech API
- **文字识别**：Tesseract.js
- **数据存储**：localStorage

## 快速开始

### 方式一：直接打开（推荐）
1. 使用浏览器打开 `index.html`
2. 开始使用，所有数据保存在浏览器本地

### 方式二：本地服务器
```bash
# 使用 Python 启动简单服务器
python -m http.server 8000

# 使用 Node.js
npx serve .

# 然后访问 http://localhost:8000
```

### 方式三：配置后台（可选）
1. 启动后端服务：`python3 main.py --start-api`
2. 在"系统设置"中启用"启用后台"开关
3. 配置后台服务地址（默认：http://localhost:8000）

## 使用说明

### 开始听写
1. 切换到"开始听写"页面
2. 勾选要听写的组别
3. 设置间隔时间和重复次数
4. 点击"开始听写"按钮

### 管理组别
1. 切换到"组别管理"页面
2. 点击"添加组别"创建新组别
3. 点击"编辑"或"删除"管理现有组别

### 使用 OCR
1. 切换到"文字识别"页面
2. 点击"开启摄像头"或"上传图片"
3. 识别完成后点击"添加到组别"

### 配置后台
1. 切换到"系统设置"页面
2. 勾选"启用后台"开关
3. 配置后台服务地址

## 注意事项

1. **语音合成**：需要网络连接下载语音数据
2. **OCR 识别**：首次使用需要下载语言包（约 10MB）
3. **数据安全**：数据存储在浏览器本地，清除浏览器数据会丢失
4. **后台模式**：需要后端服务运行才能使用
